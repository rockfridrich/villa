How It Works

This example demonstrates Villa SDK integration using plain HTML and JavaScript with no build tools.

Implementation

  • Creates an iframe pointing to https://beta.villa.cash/auth
  • Communicates via postMessage API
  • Handles authentication events (success, error, cancel)
  • Persists session in localStorage

Message Protocol

// Success
{
  type: "VILLA_AUTH_SUCCESS",
  payload: {
    identity: {
      address: "0x...",
      nickname: "user",
      avatar: { style, seed, gender }
    }
  }
}

// Error
{
  type: "VILLA_AUTH_ERROR",
  payload: { error: string, code?: string }
}

// Cancel
{ type: "VILLA_AUTH_CANCEL" }
