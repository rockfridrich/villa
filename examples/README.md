# Villa SDK Examples

Example applications demonstrating Villa SDK integration patterns.

## Available Examples

### [Next.js App](./nextjs-app)

Minimal Next.js 14 app showing complete authentication flow with Villa SDK.

**Features:**
- Passkey-based sign in/out
- Identity display with avatar
- Session persistence
- Pre-built UI components

**Quick start:**
```bash
cd nextjs-app
npm install
npm run dev
```

## Requirements

All examples require:
- Node.js 18+
- A modern browser with WebAuthn support
- HTTPS (passkeys require secure context)
